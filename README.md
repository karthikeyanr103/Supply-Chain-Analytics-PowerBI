# Supply Chain Analytics Dashboard using Power BI

## Project Overview

This project presents an interactive Supply Chain Analytics Dashboard built using Power BI. The dashboard helps analyze business performance across sales, revenue, product performance, inventory, supplier operations, manufacturing cost, logistics cost, defect rates, and customer segments.

The main goal of this project is to convert raw supply chain data into meaningful business insights using Power Query, DAX, and interactive Power BI visuals.

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Visualization
- CSV Dataset

## Dataset

The dataset contains supply chain information such as:

- Product type
- SKU
- Price
- Revenue generated
- Number of products sold
- Stock levels
- Order quantities
- Shipping costs
- Supplier details
- Manufacturing costs
- Defect rates
- Customer demographics
- Transportation modes
- Routes
- Logistics costs

## Dashboard Features

- Interactive KPI cards
- Revenue analysis by product type
- Product sales distribution
- Revenue analysis by location
- Product performance table
- Customer segment filter
- Product type filter
- Location filter
- Clean and colorful dashboard design

## Key KPIs

- Total Revenue
- Products Sold
- Profit
- Profit Margin
- Average Defect Rate
- Total Stock
- Shipping Cost
- Manufacturing Cost
- Logistics Cost

## Key Insights

- Cosmetics generated the highest revenue compared to other product categories.
- Product-wise analysis helps identify the best-performing SKUs.
- Location-based revenue analysis shows business performance across different regions.
- Sales distribution by product type gives a quick view of category contribution.
- Defect rate and stock-level tracking help monitor operational performance.

This project demonstrates practical skills in Power BI dashboard design, data cleaning, DAX measure creation, KPI reporting, and business intelligence storytelling.
